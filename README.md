# nigeria-emtl-fintech
Nigeria-compliant FinTech application for calculating Electronic Money Transfer Levy (EMTL) based on the Finance Act 2020. Designed with secure MERN architecture and national digital economy relevance.
# Nigeria EMTL FinTech System

## Project Overview
This project demonstrates a conceptual FinTech system for managing Nigeria’s
Electronic Money Transfer Levy (EMTL).

It focuses on system design, API specification, and security considerations
for EMTL calculation and transaction recording.

## Features
- EMTL levy calculation
- Transaction recording
- REST API design
- Secure system architecture

## Purpose
This project is created for learning, portfolio development, and showcasing
FinTech system design skills related to Nigeria’s financial ecosystem.
